# ✔️ App de Tareas - .NET MAUI  
**Tecnologías:** C#, .NET MAUI, SQLite, MVVM  
**Plataformas:** Android, iOS, Windows  

## 🛠️ Funcionalidades  
- Añadir/eliminar tareas.  
- Persistencia de datos con SQLite.  
- Diseño nativo con XAML.
